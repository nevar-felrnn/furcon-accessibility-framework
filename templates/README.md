# templates/

This folder contains standalone forms and checklists that support the accessibility department's operations. They are separate from the main framework chapters — you can print, share, or adapt them independently without re-assembling the full document.

## What's here

| File | Purpose |
|---|---|
| `venue-walkthrough-checklist.md` | Step-by-step checklist for assessing a venue's accessibility before the convention |
| `budget-request-template.md` | Template for submitting an accessibility budget request to convention leadership |
| `staff-training-packet-outline.md` | Outline for building a staff and volunteer training packet |
| `equipment-log.md` | Log for tracking mobility aid loans during the convention |
| `incident-report-form.md` | Form for documenting accessibility-related incidents at-con |
| `post-con-report.md` | Template for the post-convention review and lessons-learned writeup |

## How to use these

These files are included in the assembled document via `toc.md`. They are also useful on their own — print them directly or copy them into your team's shared folder.

To remove a template from the assembled document, comment out its line in `toc.md` with a `#`. To add a new one, create a `.md` file here and add its path to `toc.md`.
